# Ironclad Fitness - Premium 3D Landing Page 🏋️‍♂️

Hey there! 👋 Welcome to the repository for **Ironclad Fitness**. 

I built this project to explore how we can combine high-end aesthetic web design with scroll-linked 3D animations, similar to what you might see on an Apple product launch page. The goal was to create an uncompromising, cinematic, and luxury experience for a fitness brand.

## 🚀 Tech Stack

- **React & Vite** - For a lightning-fast development experience and optimized production builds.
- **Tailwind CSS** - For rapid, utility-first styling. I stuck to a very minimal color palette (Black, White, and Crimson Red).
- **GSAP & ScrollTrigger** - The magic behind the scroll-linked 3D marble statue animation and all the buttery-smooth text reveals.
- **Lenis** - For that premium, silky smooth scrolling feel across the entire page.

## ✨ Features

- **Scroll-Controlled Canvas Sequence**: The hero section uses an HTML5 Canvas to scrub through 240 rendered frames of a 3D Greek statue based purely on your scroll position.
- **Micro-Interactions**: Hover states, subtle lifts on cards, and custom masonry galleries designed to feel tactile.
- **Fully Responsive**: Carefully tuned to look great on desktop, tablet, and mobile devices.

## 🛠️ Setup Instructions

Want to run this locally? It's super simple. 

1. **Clone the repository:**
   ```bash
   git clone https://github.com/ShivaPrasadRaju/gym-3d-animation-website.git
   cd gym-3d-animation-website
   ```

2. **Install dependencies:**
   Make sure you have Node.js installed, then run:
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```
   Open `http://localhost:5173` in your browser, and you're good to go! 

## 📦 Build for Production

If you want to build the optimized version for production, just run:
```bash
npm run build
```
